# simple_bash
A simple shell (even though the repo is called simple_bash) implementation for my OS class.

### How to:

Compile the code:
```
make -f MakeFile
```

Try out the shell:
```
./sb_start
```

### Builtin commands:
- help
- cd \<non null path>
- ls \<nullable path>
- cat \<non null path>
- exit
- Ctrl+C signal handler (default)
